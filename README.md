# ☕ The Brew Point - Landing Page

A clean, responsive landing page for a fictional local coffee shop. This is my first project as a freelance web developer.

## 🚀 About This Project
I built this project to practice **Semantic HTML5** and **CSS3**. Coming from a background in **C and C++**, I focused on:
* **Clean Code Structure:** Using proper tags like `<nav>`, `<header>`, and `<footer>`.
* **Logical Layout:** Using CSS to center content and manage spacing (The Box Model).
* **Responsive Design:** Ensuring the menu looks good on both mobile and desktop.

## 🛠️ Tools Used
* **Code Editor:** Visual Studio Code
* **Languages:** HTML5, CSS3
